# Kull iPad Keyboard Shortcuts - Quick Reference

## Available Shortcuts

| Key Combination | Action | Description |
|----------------|--------|-------------|
| **⌘ N** | New Shoot | Start a new photo culling session |
| **⌘ ,** | Settings | Open app settings |
| **⌘ R** | Refresh | Refresh credit balance |

## How to Use

1. **Connect Keyboard**
   - Attach Magic Keyboard, Smart Keyboard, or Bluetooth keyboard to your iPad
   - Ensure keyboard is paired and active

2. **View Available Shortcuts**
   - Press and hold **⌘ (Command)** key
   - iOS will display an overlay showing all available shortcuts

3. **Execute Shortcuts**
   - Press **⌘** + the corresponding letter key
   - Action will execute immediately

## Tips

- Shortcuts work from any view in the app
- Hold ⌘ to see discoverability overlay
- Shortcuts are case-insensitive (⌘N = ⌘n)
- Shortcuts work in landscape and portrait orientations

## Compatibility

- **iPad Models:** All iPad models with iOS 17.0+
- **Keyboards:** Magic Keyboard, Smart Keyboard, Bluetooth keyboards
- **iOS Version:** 17.0 or later
